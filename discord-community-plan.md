# Discord Community Setup Plan
**Status:** Ready to execute (Week 4: Feb 24-26)  
**Purpose:** Community for automation/AI builders interested in OpenClaw, agent infrastructure, etc.

---

## Server Structure

### Channels

**#announcements**
- Automated posts: Weekly digest of @stanleybodewell posts
- Manual: Casey shares major updates
- Locked (Casey posts only)

**#automation-demos**
- Stan posts daily automation showcases
- Community members share their builds
- Discussion: "How would you solve this?"

**#ask-stan**
- Community members ask Stan questions
- Stan responds with research + advice
- Q&A thread format

**#general**
- Off-topic, community chat
- Casual conversation

**#voice** (optional)
- Voice channel for live conversations
- Stan can share audio summaries here (once TTS works)

---

## Roles

**@Admin** (Casey)
- Full server permissions
- Can moderate, mute, kick
- Manage channels, announcements

**@Moderator** (optional: invite later)
- Can moderate conversations
- Can manage threads

**@Community**
- Everyone else
- Can post, react, ask questions

---

## Stan's Role

**Automated:**
- Post daily automation showcase (text)
- Post TTS summary (audio, once ready)
- Monitor #ask-stan for questions
- Respond to questions with research + references

**Manual:**
- Casey decides what Stan announces
- Stan never posts promotional content
- Stan never solicits signups
- Community is about sharing & learning

---

## Setup Steps

1. **Create Discord Server**
   - Name: "Automation Builders" or "OpenClaw Community" (Casey decides)
   - Region: US (for latency)

2. **Create Channels** (above list)

3. **Create Roles** (above list)

4. **Invite Stan as Bot**
   - Generate Discord bot token
   - Add bot to server
   - Configure permissions

5. **Configure Stan's Permissions**
   - Can post in #announcements (for scheduled posts)
   - Can post in #automation-demos (daily showcase)
   - Can read #ask-stan (to respond)
   - Cannot delete/moderate (human moderation only)

6. **Set Welcome Message**
   - Explain what server is about
   - Point to #ask-stan for questions
   - Link to @stanleybodewell Twitter
