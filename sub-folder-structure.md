# SUBCONTRACTOR FOLDER MANAGEMENT SYSTEM
## Organize Your Subs for Legal Protection & Easy Reference

---

## THE SYSTEM

Create **one folder per subcontractor** with the following structure. This discipline is what separates GCs who get sued from GCs who don't.

**Physical or Digital?** Use whichever you prefer, but be **consistent**. Digital is easier for photos and scanning, but physical shows a judge you took it seriously.

---

## FOLDER NAMING CONVENTION

Use this format to keep subs organized by project:

```
[PROJECT ADDRESS]_[SUB TRADE]_[SUB NAME]
```

**Examples:**
- `123MainSt_Plumbing_JohnDoeElectrical`
- `456OakAve_Framing_RiversideConstruction`
- `789PineRd_Electrical_SmithElectric`

**Benefits:**
- Easy to find sub by project
- Trade is immediately clear
- Name prevents confusion if multiple plumbers (etc.) work for you

## REQUIRED DOCUMENTS IN EACH FOLDER

### **TIER 1 — INSURANCE & LEGAL (CRITICAL)**

**MUST have before work starts:**

1. **Certificate of Insurance (COI) — ORIGINAL**
   - Copy of the signed ACORD form from their insurance agent
   - Store alongside this document: expiration date, renewal reminder date
   - Keep copies from every renewal during project

2. **Workers' Compensation Documentation — ORIGINAL**
   - **If they have employees:** Current WC certificate showing coverage
   - **If sole proprietor:** Signed Kansas Department of Labor **Form K-WC 110** (Exemption)
   - Keep on file for statute of limitations (3–5 years after job completion)

3. **Signed Subcontractor Agreement**
   - Use template provided (subcontractor-agreement-template.md)
   - Both parties signed and dated
   - Include any project-specific amendments or change orders

4. **W-9 Form**
   - Required for any sub paid $600+ in a calendar year
   - Get this before first payment
   - Store and reference for 1099-NEC reporting

5. **Contractor License Copy** (if trade requires licensing)
   - Kansas trades requiring licenses: Electrical, Plumbing, HVAC, Roofing (in some jurisdictions)
   - Copy of current, valid license
   - Note license expiration date — follow up for renewal proof if license expires during project

---

### **TIER 2 — PROJECT DOCUMENTS**

1. **Scope of Work / Project Specifications**
   - Detailed description of what sub is doing
   - Reference plans (can be photos or PDF)
   - Any industry standards they must follow

2. **Payment Invoice & Payment Records**
   - Copy of all invoices received from sub
   - Proof of payment (canceled check, ACH transfer receipt, etc.)
   - Track what has been paid and what balance remains

3. **Change Order Forms** (if any)
   - Signed change orders with any scope, timeline, or price changes
   - Keep all change orders together chronologically
   - Note which invoices correspond to original scope vs. changes

4. **Project Photos**
   - Before photos of area worked on
   - During photos showing sub's work
   - After/completion photos
   - Any defect photos (if corrections needed)
   - Date photos and store in subfolder: `/[ProjectName]_Photos/`

---

### **TIER 3 — COMMUNICATION & CORRESPONDENCE**

1. **Email Chain with Sub**
   - Print or export emails regarding this sub
   - Include contract negotiation, scheduling, questions, etc.
   - Timestamps matter — they prove when you knew something

2. **Text Messages** (if you text back and forth)
   - Screenshot and print or save as PDF
   - Include date/time stamps
   - Useful proof of scope changes or agreed modifications

3. **Phone Call Notes**
   - If you discuss important items, write a follow-up email: "Per our call today, we agreed [X]. Please confirm."
   - File the email
   - Creates a paper trail

---

### **TIER 4 — QUALITY & COMPLIANCE**

1. **Inspection Reports**
   - City/county inspection reports for work sub completed (if applicable)
   - Passed inspections = good evidence sub did code-compliant work
   - Failed inspections = document sub's corrections

2. **Lien Waiver**
   - Final **signed lien waiver** from sub before final payment
   - Confirms all work complete and sub won't file a lien against property
   - Use Kansas statutory form or attorney-reviewed template

3. **Warranty Documentation**
   - If sub provides any warranty on materials or workmanship, attach it
   - Note warranty period and coverage details
   - Keep for customer reference and sub follow-up if issues arise

3. [ ] Receive and file COI → verify using COI checklist (coi-checklist.md)
4. [ ] Receive and file W-9
5. [ ] Prepare signed subcontractor agreement
6. [ ] Both of you sign agreement
7. [ ] File signed agreement
8. [ ] Give sub start date
9. [ ] Take before photos → file in 05_Photos
10. [ ] Monitor project → file photos, invoices, emails, correspondence
11. [ ] Upon completion, request lien waiver → file
12. [ ] Final payment only after lien waiver received
13. [ ] Store folder for 3–5 years (statute of limitations on construction defects)

## KEY RULES

### Rule 1: Nothing Starts Without Documentation
- No COI? They don't work.
- No signed agreement? They don't work.
- No W-9? They don't work (or can't pay until you have one).

### Rule 2: Everything Gets Filed
- Every email about the sub = print it and file it
- Every photo = dated and filed
- Every invoice = filed with payment proof
- Every phone call with important info = follow-up email filed

### Rule 3: Folder is Your Defense
If there's ever a dispute (sub claims nonpayment, homeowner claims bad work, injury happens), your folder answers:
- Who was on the job (licensing)
- When they worked (photos, invoices, timeline)
- What they were supposed to do (scope of work)
- How much they were paid (invoices + payment proof)
- Were they insured? (COI)
- What happened? (photos, inspections)

**A complete folder = documented evidence that you managed the job properly.**

### Rule 4: Keep Folders After Project Completion
**Don't throw them away.** Keep for:
- **3 years:** Kansas standard warranty period for construction work
- **5 years:** Kansas statute of limitations for mechanic's liens
- **7 years:** IRS 1099 documentation

If a homeowner calls 2 years later saying "That electrical work is failing," your folder proves you hired a licensed electrician who carried insurance and passed inspection.

## SAMPLE EMAIL TO SUB (Documentation Request)

---

**Subject: Before You Start — Required Documentation**

Hi [Sub Name],

Before you can begin work on [Address], I need the following in my file. Please send these by [DATE]:

1. **Certificate of Insurance** (ACORD form) showing:
   - [YOUR BUSINESS NAME] as Additional Insured
   - Minimum $1M General Liability
   - Waiver of Subrogation
   - 30-day cancellation notice

2. **Workers' Compensation** — either:
   - Current WC certificate with coverage amounts, OR
   - Signed Kansas K-WC 110 exemption form (if sole proprietor with no employees)

3. **W-9 Form** (for 1099 reporting)

4. **Copy of Kansas Contractor License** (if your trade requires one)

Once I have all of these, I'll send you the signed subcontractor agreement for both of us to sign.

This protects both of us and keeps the project clean. Let me know if you have questions.

Thanks,  
[Your Name]  
[Phone]

## USING THE FOLDER SYSTEM

**Each week during project:**
- Check folder — is documentation up to date?
- Add any new photos, invoices, or communications
- Check insurance expiration dates
- Flag any concerns (slow progress, quality issues, missed deadlines)

**At project completion:**
- Get lien waiver (must be signed before final payment)
- Take final photos
- File final invoice and payment
- Update folder status to "Complete"
- Move to archive or year-end storage

**If a dispute arises:**
- Pull the folder
- Review timeline, scope, payments, photos
- Use documentation to defend your position

---

This system takes 15 minutes per sub to set up and 5 minutes per week to maintain. It's the difference between "I remember it happened" and "Here's proof."
